# DOSCO Godot 4 - Package Complet

Ce dossier contient une version améliorée et structurée du projet DOSCO - La Bataille des Étoiles.

## Fichiers clés inclus

- README.md (Documentation principale)
- DOSCO_Release_Roadmap.md (Feuille de route pour la release)
- src/core/AI.gd (IA optimisée)
- src/ui/StarPiece.gd (Pièce avec animations)
- scenes/effects/CaptureEffect.gd (Effet de capture)

## Comment utiliser

1. Copier le contenu dans un projet Godot 4.3+
2. Compléter les scènes .tscn manquantes en suivant les descriptions dans les scripts
3. Configurer les icônes et le keystore pour l'export Android
